# Lista-de-Tareas-To-Do-List-
Lista de Tareas (To-Do List)

Este es un pequeño proyecto en HTML, CSS y JavaScript donde puedes agregar tareas a una lista y eliminarlas simplemente haciendo clic sobre ellas. Es un ejercicio ideal para practicar manipulación del DOM, eventos y estilos básicos.

Funcionalidades

Agregar tareas nuevas mediante un campo de texto.

Eliminar tareas haciendo clic sobre ellas.

Mensaje de alerta si intentas agregar una tarea vacía.

Animación al pasar el ratón por las tareas.

Diseño limpio y centrado usando HTML + CSS puro.

Tecnologías utilizadas

HTML5: estructura de la página

CSS3: estilos, sombras, bordes, animaciones (hover)

JavaScript: manejo del DOM, eventos, creación y eliminación dinámica de elementos

Conceptos que se manejan en este proyecyto:

document.getElementById()

addEventListener()

createElement() y appendChild()

Eventos delegados mediante e.target

Validación básica de formularios

Estilos con hover, box-shadow, border-radius, transform


Una pequeña aplicación centrada en la pantalla que permite escribir una tarea y verla aparecer en una lista ordenada. Cada tarea se puede borrar fácilmente haciendo clic sobre ella.

Cómo usarlo

Escribe una tarea en el campo de texto

Pulsa el botón Agregar

Haz clic sobre cualquier tarea para eliminarla

Objetivo del proyecto

Este proyecto sirve como base para avanzar en frontend, practicando:

DOM dinámico

Interacción usuario-interfaz

Pequeñas animaciones y efectos
